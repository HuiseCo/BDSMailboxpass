# BDSMailboxpass- BDS邮箱通行证
---
## 原理:❓
1. 服务端启动插件  
2. 玩家进入游戏侦测其账号是否绑定过邮箱(是-跳转第六步)  
3. 输入邮箱响应回服务端  
4. 通过邮箱服务商提供的开放接口发送随机验证码给用户邮箱  
5. 玩家输入验证码  
6. 进入游戏
